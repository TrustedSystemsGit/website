# TrustAI FPGA Website (Modern Design)

Modern static site for the startup.

## Features
- Responsive grid/flex layouts
- Dark/light theme with persistence
- Animations and gradients
- Chart.js integration
- Form with validation
- SEO meta tags
- Google Analytics GA4
- Sitemap.xml and robots.txt for search engines

## Deployment
1. Clone repo.
2. Replace GA tracking ID (G-XXXXXXXXXX) with yours.
3. For form: Replace Formspree action with your endpoint/API key.
4. Host on GitHub Pages/Netlify: Push to main, set Pages branch. Ensure sitemap/robots in root.

## Customization
- Fonts: Roboto via Google Fonts
- Images: Replace placeholders
- JS: Extend for more interactivity
- Update sitemap dates/URLs with your domain
